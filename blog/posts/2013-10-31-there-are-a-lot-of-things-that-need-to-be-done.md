date: 2013-10-31
title: "There Are a Lot Of Things That Need To Be Done"
---



There are a lot of things that need to be done. But the only reason they need
to be done *fast* is because *I* need them to be done fast. I need to work at
a frantic pace because I feel more safe when I'm working and not stopping.

Things that need to be done can optionally be done at a healthy and relaxed
pace. They'll still get done, but I won't get my fix. Good and bad are
intertwined, and I lack the wisdom to discern between them. That's not a good
excuse for working this way. It's reason to stop completely, and sort out the
truth from the lies… then resume the work from a healthy and correct
perspective.

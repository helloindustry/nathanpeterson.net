<p>I've had some people ask me about how we do our click tracks with <a href="http://helloindustry.com/" target="_blank">Hello Industry</a> and at <a href="https://www.richwoods.org/" target="_blank">Richwoods</a>. Hope this helps.</p>&#13;
<p><strong>TOOLS</strong></p>&#13;
<p>We use <a href="https://www.ableton.com/live-8" target="_blank">Ableton Live</a> to run our click. If a simple click is all you'll ever need, you may want to look at something more simple like a <a href="https://www.amazon.com/BOSSS-DB-90-BOSS-Metronome/dp/B000ATOFS4/ref=sr_1_1?ie=UTF8&amp;qid=1337205763&amp;sr=8-1" target="_blank">Dr Beat</a>. But using Live gives us the ability to also run loops (backing tracks) and even runs our light show (another post).</p>&#13;
<p>We run Ableton from a macbook and output using the built-in audio port. Nothing fancy.</p>&#13;
<p>We run a <a href="https://www.amazon.com/Hosa-CMP159-Feet-Speaker-Cable/dp/B005HGM1D6/ref=sr_1_1?ie=UTF8&amp;qid=1337205959&amp;sr=8-1" target="_blank">stereo 1/8 to L/R 1/4 cable</a> from the laptop to a stereo direct box. We use a <a href="https://www.amazon.com/Radial-ProD2-Passive-Channel-Direct/dp/B000B85FV0/ref=sr_1_1?s=musical-instruments&amp;ie=UTF8&amp;qid=1337205978&amp;sr=1-1" target="_blank">radial DI</a> which has served us well for years.</p>&#13;
<p><strong>SIGNAL FLOW</strong></p>&#13;
<p>In order to avoid adding an external interface, we keep our loops/tracks mono. That lets us output the click and count off to the right channel and send that only to our in-ears. Then we send the left channel ("L-eft for L-oop" is easiest to remember) to the FOH console the same way we send any other instrument. * Always test this before a show. We almost always plug in the cables backwards!</p>&#13;
<p><strong>USAGE</strong></p>&#13;
<p>We set up a template which we start with for every show. The template includes a 2-bar count off for every song. So we hear a lady say "1 2 3 4 1 2 3 4" and then we start the song. We set up shortcuts to each song so that our keyboard player (or drummer) only has to hit one key on the keyboard to trigger each song.</p>&#13;
<p>Each scene (row) in Live is a different song. By changing the title of each scene in the Master column, you tell Live to change its tempo for that song. This lets us cut and paste songs and just change the BPM to get the desired tempo of click track. So essentially, every song without a loop is identical.</p>&#13;
<p>Rather than try to explain every detail of this, I'm attaching a couple of screen shots and a blank template I made up, including the count off and click wav files. You should be able to load this up in Ableton 8 and start using it right away. I made sure it was less than 6 songs so that it will still load in Ableton Lite (Lite has a 6-scene limitation). Try it out and let me know if you have any questions.</p>&#13;
<p>Have fun!</p>&#13;
